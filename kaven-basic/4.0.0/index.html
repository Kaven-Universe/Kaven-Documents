Kaven-Basic

Installing

 
# NPM
npm install kaven-basic

# Yarn
yarn add kaven-basic

# PNPM
pnpm i kaven-basic

Usage

Node.js:

import { GenerateGuid } from "kaven-basic";

console.log(GenerateGuid());
